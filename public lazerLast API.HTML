Публичное API LazerLast

Это публичное API к данным сервера на проекте LazerLast.

Данное API позволит вам получить информацию об игроках с помощью различных команд.

описание команд и примеры применения можно найти тут:

Команды к API и примеры к ним

Описание Команда Аргументы Пример применения

Получение данных на Prison

prison.php

?player=Игрок

&format=формат(text/JSON)

prison.php?player=_GameDoctor_&format=json
